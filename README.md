# CGI GenAI Ops Solution Architect — Independent Candidate Campaign

A public candidate vision by Russell Dudek for CGI's GenAI Ops Solution Architect role (J0626-1398).

## Thesis

**Make enterprise AI load-bearing.** Enterprise AI scales when every use case enters through an architecture that binds business consequence, a reusable pattern, data and identity boundaries, evaluation, observability, human authority and production ownership before production pressure exposes the gaps.

## Signature experience

The hero is a guided **Architecture Load Test**. Four keyboard-operable use cases move through eight explicit enterprise layers. The complete path is tested once per scenario; unresolved controls remain attached to the failed layer, and the final Permit, Conditional or Hold disposition appears only after the full architecture has been inspected.

The supporting **Enterprise AI Building Code** and **Architecture Permit Review** translate platform governance into a visible operating mechanism rather than a generic architecture claim.

## Campaign surfaces

The repository contains the candidate vision site, exact two-page résumé, exact one-page cover letter, two-page interview thesis brief, one-page landscape 90-day entry plan and one-page landscape Architecture Permit. Every printable artifact has a committed PDF and visible download route.

## Publication

The static site uses repository-relative paths and includes a GitHub Pages deployment workflow. CGI's official identity is used only to identify the intended employer. This is independent candidate work and is not sponsored or endorsed by CGI.
